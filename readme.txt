Software Installations:
- Install JDK-1.7+
- Install IntelliJ Idea community edition 13.1.3 (do not install older version than this)
- From within Idea, install latest Scala plugin

- Install git (latest)
- Clone this repo: git clone https://github.com/mushtaq/scala-basics-2.git
- cd scala-basics-2
- ./sbt gen-idea (this will take a few minutes to complete)
- From within idea, open project and select scala-basics-2 folder

- Install scala 2.10.3 from http://www.scala-lang.org/download/2.10.3.html
- Install SourceTree from http://www.sourcetreeapp.com/ (optional)



For JDK1.7:
sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-jdk7-installer

For IntelliJ 13.1.3 community edition fnd the jar ideaIC-13.1.3.tar.gz.
